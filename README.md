# 🏋️‍♂️ Gym Power Flex

Un sitio web moderno para un gimnasio, diseñado con enfoque visual, estructura clara y experiencia de usuario intuitiva.
Este proyecto representa un avance en el desarrollo frontend utilizando HTML y CSS puro.

---

## 🎯 Objetivo del proyecto

Desarrollar la estructura inicial de una página web para un gimnasio, aplicando buenas prácticas de maquetación, diseño responsive básico y organización del código.

---

## 🧩 Secciones implementadas

* 🏠 **Inicio**

  * Navbar fijo (sticky)
  * Carrusel de imágenes (Bootstrap)
  * Sección de beneficios
  * Galería visual

* 💳 **Planes**

  * Tarjetas de precios
  * Diseño con grid responsive

* 🏋️ **Clases**

  * Catálogo de actividades
  * Imágenes optimizadas

* 📩 **Contacto**

  * Formulario funcional (estructura HTML)
  * Diseño limpio y centrado

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap 5 (solo en la página principal)

---

## 🎨 Características destacadas

* Navbar fijo con diseño consistente
* Uso de Flexbox y Grid
* Diseño limpio y moderno
* Organización modular por páginas
* Efectos hover en tarjetas
* Imágenes adaptables con `object-fit`

---

## 🧠 Aprendizajes

Durante el desarrollo de este proyecto se trabajó en:

* Maquetación estructurada (header, main, section, footer)
* Control de layouts con Flexbox y Grid
* Manejo de inconsistencias entre páginas
* Integración parcial de Bootstrap
* Mejora visual mediante sombras y espaciado
